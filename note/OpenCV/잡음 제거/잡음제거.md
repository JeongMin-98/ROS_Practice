# 영상의 잡음

영상의 픽셀 값에 추가되는 원치 않은 형태의 신호다. 
> 카메라에서 광학 신호를 전기적 신호로 변환하는 과정에서 잡음이 생성될 수 있다. 

## 잡음의 종류
+ 소금후추노이즈(salt and pepper noise)
  + 영상에 흰점, 검은점이 뿌려져있듯이 표현되어있는 것
+ 가우시안노이즈(guassian noise)
  + 가우시안 분포에 따라 노이즈가 영상에 표현되어있는 것


# 잡음 분석
## 프로파일
영상에서 특정 경로 상에 있는 픽셀의 밝기 값을 그래프 형태로 나타낸 것
