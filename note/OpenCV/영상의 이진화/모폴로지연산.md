# 모폴로지

영상을 형태학적인 측면에서 다루는 기법
+ 전처리 또는 후처리 형태로 널리 사용됨
+ 수학적 모폴로지

+ 범용 모폴로지 연산 함수

```cpp
morphologyEx
```

## 구조요소
+ 모폴로지 연산의 결과를 결정하는 커널, 마스크

```cpp
getStructuringElement
```

## 침식
+ 구조 요소가 객체 영역 내부에 완전히 포함될 경우 고정점 픽셀을 255로 설정
+ 침식 연산은 객체 외곽을 깍아내는 연산이다. 
+ 객체의 크기는 줄어들고 배경은 확대된다.
+ 구조 요소를 가지고 전체 영상에 대해 scaling한다. 

```cpp
erode
```

## 팽창
+ 구조 요소와 객체 영역이 한 픽셀이라도 만날 경우 고정점 픽셀을 255로 설정
+ 객체의 외곽 영역을 확대시키는 연산

```cpp
dilate
```

## Opening (열기)
침식 연산을 먼저 수행한 이후에 팽창 연산을 수행한다. 

## Closing (닫기)
팽창 연산을 먼저 수행한 이후에 침식 연산을 수행한다.
