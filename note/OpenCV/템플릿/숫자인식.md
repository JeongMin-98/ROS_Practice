## 숫자 인식
> 템플릿 매칭을 이용하여 숫자를 인식하는 프로그램 만들기

+ 인식이란?
  + 여러 개의 클래스 중에서 가장 유사한 클래스를 선택

> 컴퓨터에서는 유사도 함수를 통해서 인식한다. 


### 숫자 템플릿 영상 생성
+ Consolas 폰트로 쓰여진 숫자 영상을 digit0.bmp~digit9.bmp로 저장
+ 각 숫자 영상의 크기는 100 X 150 크기로 정규화한다. 

### 템플릿 매칭을 이용한 숫자 인식
+ 인쇄체 숫자 인식 과정
+ 숫자 영상이 검정색으로 되어있기 때문에 inverse시켜준다.
