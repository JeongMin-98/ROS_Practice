# 영상의 명암비 조절
## 명암비
+ 밝은 곳과 어두운 곳 사이에 드러나는 밝기 차이 정도의 차이
+ 컨트라스트, 대비
### 명암비 조절 방법
입력 영상의 각 픽셀에 일정 비율의 상수를 곱하는 형태이다.
> 이미지의 밝기 조절과는 다르게 상수를 더하지 않고 곱한다. 
+ 그리고 영상 자료형의 범위를 넘어가지 않도록 saturate함수를 적용시켜준다.
+ 이러한 이유로 변환 함수의 형태가 상수배를 하는 경우 명암비가 제대로 적용되지않는 문제가 발생한다. 
#### 효과적인 명암비 조절
+ alpha값을 명암비의 강도를 조절한다. 
+ 0~255의 중간값인 128을 구한다. 
+ 해당 값에 해당하는 픽셀값은 128로 반환한다. 
```cpp
float alpha = 1.0f;
Mat dst = saturate_cast<uchar>(src + (src - 128) * alpha);
```
#### 평균 명암비 조절
+ 원본 영상 픽셀의 평균값을 구한다. 
+ 평균값에 해당하는 값은 평균값을 반환하도록 한다. 
```
float alpha = 1.0f;
int m = (int)mean(src)[0];
Mat dst = src + (src - m) * alpha;
```
