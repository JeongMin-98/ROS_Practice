# 영상의 산술 연산
## 덧셈
```cpp
void add(InputArray src1, InputArray src2, OutputArray dst,
                      InputArray mask = noArray(), int dtype = -1);

```
## 뺄셈
```cpp
void subtract(InputArray src1, InputArray src2, OutputArray dst,
                           InputArray mask = noArray(), int dtype = -1);
```
+ 인자
  + src1: 첫 번째 입력 행렬 또는 스칼라
  + src2: 첫 번째 입력 행렬 또는 스칼라
  + dst: 출력 행렬, dst의 깊이는 src1, src2의 깊이와 같거나 또는 dtype 인자에 의해 결정됨.
  + mask: 마스크 영상, mask 행렬 원소 값이 0이 아닌 위치에서만 연산을 수행함
  + dtype: 출력 행렬의 깊이

## 덧셈 연산
두 영상의 같은 위치에 존재하는 픽셀의 값을 더하여 결과 영상의 픽셀 값으로 설정한다. 그리고 해당 연산은 그레이스케일 영상의 경우 255보다 크면 픽셀 값을 255로 설정하고 0보다 작으면 0으로 설정한다.

## 가중치 합
> 포화 연산이 많이 생성되는 덧셈 연산보다는 가중치를 적용하는 덧셈 연산을 더 선호한다. 

두 영상의 같은 위치에 존재하는 픽셀 값에 대하여 가중합을 계산하여 결과 영상의 픽셀 값으로 설정한다. 보통 alpha+beta = 1이 되도록 설정함으로 두 입력 영상의 평균이 유지된다.
+ alpha와 beta값을 조정하면 영상에 Dissolve를 구현할 수 있다.

### 평균 연산의 응용
+ 촬영할 때마다 노이즈(잡음)이 생기는데 촬영한 이미지의 개수를 나눠주게 되면 잡음이 없고 선명한 영상을 얻을 수 있다. 


## 뺄셈 연산
두 영상의 같은 위치에 존재하는 픽셀 값에 대하여 뺄셈 연산을 수행하여 결과 영상의 픽셀 값을 설정한다. 그리고 포화연산을 적용한다. 
덧셈 연산과 달리 뺄셈 연산은 영상의 순서에 따라 결과가 달라진다.

## 차이 연산
두 입력 연산에 대하여 뺄셈 연산을 수행한 후 그 절댓값을 이용하여 결과 영상을 생성하는 연산
+ 주로 서로 다른 영상에 대해 변화를 감지하는 데 이용한다.
